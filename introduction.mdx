## What you can build

The Kintone developer platform gives you two complementary APIs:

**JavaScript API** — Add custom behavior directly inside Kintone apps. Register event handlers that run when records are created, edited, or deleted. Read and modify field values, show or hide UI elements, and make API requests from within your customization code.

**REST API** — Integrate Kintone with external systems. Create, read, update, and delete records programmatically. Manage app settings, form fields, views, permissions, spaces, and plug-ins from any HTTP client.
